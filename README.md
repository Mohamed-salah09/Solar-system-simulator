# 🌌 Solar System Simulator

A C++ project that simulates the motion of planets around the Sun and visualizes orbital movement in real time.

## Features

* Planetary orbit simulation
* Real-time animation
* Multiple planets with different orbital speeds
* Object-oriented design
* Expandable project structure

## Purpose

This project is designed to explore simulation development, mathematical modeling, and software design concepts through a solar system visualization.

## Future Improvements

* Additional planets and moons
* Realistic orbital mechanics
* Camera controls and zooming
* Planet information panels
* Enhanced visual effects
