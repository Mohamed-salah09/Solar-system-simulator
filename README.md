 # 🌌 Solar System Simulator

A real-time Solar System simulation built with C++ and SFML, designed to visualize planetary motion, orbital mechanics, and object-oriented software design principles.

## 🚀 Overview

Solar System Simulator is an interactive graphical application that recreates the motion of planets orbiting the Sun. Each planet follows its own orbital path with unique speed, size, and visual appearance, creating a dynamic representation of our solar system.

This project was developed to strengthen skills in:

* C++ Programming
* Object-Oriented Design (OOP)
* Mathematical Modeling
* Real-Time Graphics Rendering
* SFML Game Development
* Software Engineering Practices

## ✨ Features

* ☀️ Central Sun rendering
* 🪐 Eight planets with individual properties
* 🔄 Real-time orbital movement
* 🎯 Object-oriented architecture
* 🌍 Planet labels and identification
* 🌌 Dynamic starfield background
* ⭕ Orbit path visualization
* 💍 Saturn ring rendering
* ⚡ Frame-independent animation using Delta Time
